--- layout: docs title: "Migrate from Vafpress" ---

Steps

Replacing Vafpress with FavPress is easy.
  1. Depend on FavPress instead Vafpress
  2. Replace the old prefix vp used by Vafpress with favpress.
  3. Replace the old prefix VP used by Vafpress with FavPress.

Example

Old Vafpress code


    $theme_options = new VP_Option(array(
        ...
    ));

    $option = vp_option('option');
    $metabox = vp_metabox('metabox');

New FavPress code


    $theme_options = new FavPress_Option(array(
        ...
    ));

    $option = favpress_option('option');
    $metabox = favpress_metabox('metabox');
